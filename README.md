# dslinjector (discord-slash-injector)
The post command data injector

## How to Use

1. Clone source
```shell
git clone https://github.com/DevProject04/dslinjector.git
```

2. Build source
```shell
make
```

3. Install source
```shell
make install
```

4. Run program
```shell
dslinjector <guild_id> <client_id> <token> <json_data>
```
